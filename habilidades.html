Ecossistema Front-end & UI

  • React & Vite: Construção de interfaces dinâmicas, componentização e uso de ferramentas de build de alta velocidade.

  • Tailwind CSS & Shadcn/UI: Estilização moderna baseada em classes utilitárias e componentes prontos.

  • React Router Dom: Gerenciamento de rotas dinâmicas, permitindo navegação sem recarregar o navegador.

  • Bibliotecas de Interface: Experiência com Lucide React (ícones vetoriais) e Sonner (notificações em tela).

Linguagens de Programação e Marcação

  • Python: Fundamentos sólidos em lógica de programação. Capacidade de criar scripts básicos para automação e desenvolver servidores backend simples (como a API de e-mail deste portfólio).

  • TypeScript (TSX): Experiência prática inicial na estruturação de interfaces com React, utilizando tipagem básica para entender como prevenir bugs e organizar melhor o código.

  • HTML5 & CSS3: Compreensão prática de marcação estruturada e estilização visual. Capaz de construir layouts responsivos e aplicar designs modernos do zero.

  • JSON: Familiaridade com o formato para leitura de dados e configurações básicas de ambiente de projetos (como em arquivos package.json e vercel.json).

Ambiente, Build & Deploy

  • Node.js: Experiência no uso do ambiente de execução e gerenciamento de pacotes (npm) para instalar e rodar bibliotecas.

  • Vercel CLI: Experiência prática em deploy de aplicações na nuvem e configuração de rotas via linha de comando.

  • Git e GitHub: Controle de versão para salvar projetos e organizar repositórios de código.

Suporte e Infraestrutura

  • Manutenção de Computadores: Experiência prática em montagem, diagnóstico e manutenção preventiva de hardware (limpeza, troca de pasta térmica, fontes e memórias). Formatação e configuração de sistemas Windows 10 e distribuições Linux (Xubuntu).

  • Suporte Técnico (N1/N2): Resolução de incidentes de TI, auxiliando usuários na instalação de impressoras em rede, configuração de certificados digitais, problemas em navegadores e instalação de softwares.

  • Gestão de Chamados: Utilização do sistema GLPI para administrar a abertura, acompanhamento e fechamento de chamados técnicos.

Administração de Sistemas

  • Microsoft 365 Admin Center: Gerenciamento de ambiente corporativo, criação de usuários, atribuição de licenças, configuração de caixas de correio compartilhadas e criação de listas de distribuição.

  • Active Directory: Administração de identidades e acessos. Criação de novos usuários baseados em perfis espelhos (políticas de grupos) e desativação segura de contas inativas.

Banco de Dados

  • Modelagem e Estruturação (MySQL): Conhecimento prático na criação de bancos de dados relacionais para projetos acadêmicos e ferramentas internas. Experiência na elaboração de Minimundo, Modelagem Entidade-Relacionamento (MER) e Modelagem Relacional (MR).

  • Desenvolvimento SQL: Capacidade de criar scripts DDL para estruturação de tabelas e DML para inserção e manipulação de dados. Conhecimento aplicado na extração de informações utilizando JOINs, Subselects e Views.

  • Regras de Negócio: Compreensão e aplicação inicial de Procedures, Functions e Triggers para automatizar rotinas e manter a integridade dos dados.

UI/UX e Comunicação

  • Prototipação (Figma): Criação de interfaces de usuário (UI) e protótipos navegáveis para estruturar a experiência do usuário (UX) antes do desenvolvimento do código.

  • Marketing Digital Corporativo: Criação de materiais gráficos e identidade visual utilizando Canva para redes sociais. Utilização de Inteligência Artificial (Prompt Engineering) para auxiliar no copywriting.

Competências Profissionais
